2. var = input('hint string'): show hint string, read variable from keyboard then store to the variable 變數=輸入('字串'):顯示字串，儲存和標記程式內資訊

Homework 2: Write a program / some programs which use these operator
常見的運算子 (Operator)
+: 常見的加法有兩種，數字(整數或是浮點數)加法，字串加法
-: 數字(整數或是浮點數)減法
*: 常見的乘法有兩種，數字(整數或是浮點數)乘法，字串乘法
/: 數字(整數或是浮點數)除法
%: 取餘數，念作 mod; 例如判斷是不是倍數關係，或是判斷奇數偶數
**: 取次方

變數名稱規則
1. 只能是一個字詞，不能有空格
2. 只能用英文字母、數字、底線，不能用連字符號-、特殊符號(如 $、'')
3. 不能以數字開頭，不能只有數字
